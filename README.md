# WordWeave 📖✨

> **把枯燥的单词表，变成你追着读的英文故事。**
> **或者——切到语法模式，让 AI 用地狱笑话教你复杂句型。**

[🚀 在线使用](https://tntsg1.github.io/WordWeave/) · 零安装 · 纯浏览器运行

---

## 🎯 这是什么

背单词最痛苦的是什么？打开单词书，abandon, abandon, abandon……三分钟就困了。

WordWeave 换个思路：**让 AI 用你要背的单词写故事。** 你上传一个单词表（一行一个），每次刷新，AI 随机抽词现场编一篇全新英文短篇——目标词橙色加粗，中文释义紧随其后。**每一个单词都可以点击看释义。**

右上角一键切换到 **语法模式**——复杂句型 + 十岁小孩都看得懂的简单词汇 + 地狱笑话/黑色幽默风格，练语法不再枯燥。

你读的是故事，记的是单词，练的是语法。

## ✨ 功能

### 📝 单词故事模式（默认）

| 功能 | 说明 |
|------|------|
| 📂 上传词库 | 拖放 `.txt` 文件，一行一个单词，自动入库 |
| 🔀 多词库管理 | 下拉切换、上传新词库、删除旧词库 |
| 📖 AI 编故事 | 随机抽 12 个单词，DeepSeek 现场写短篇 |
| 🟠 目标词高亮 | 橙色加粗，点击弹窗看释义 |
| 🔤 中英对照 | 每个单词后跟绿色中文释义，**所有单词均可点击** |
| 🇨🇳 全文翻译 | 底部完整中文译文 |

### 🧠 语法实战模式（右上角切换）

| 功能 | 说明 |
|------|------|
| 💀 地狱笑话 | 葬礼乌龙、医疗事故、职场丑闻……黑色幽默 |
| 📐 复杂语法 | 虚拟语气、倒装、条件句、强调句、分词短语 |
| 🗣️ 简单词汇 | 刻意限制在十岁小孩词汇量，复杂度来自句型而非单词 |
| 📋 语法解析 | 每篇下方列出用了哪些语法结构 + 原文例句 |
| 🇨🇳 中文翻译 | 全文翻译 |
| 🔄 独立刷新 | 不看单词的时候，专门来这里练阅读 |

## 🚀 怎么用

### 方式一：在线（推荐）

直接打开 👉 **[tntsg1.github.io/WordWeave](https://tntsg1.github.io/WordWeave/)**

首次使用输入 DeepSeek API Key（在 [platform.deepseek.com](https://platform.deepseek.com/api_keys) 免费注册获取）。Key 只存在你的浏览器里，不上传任何服务器。

### 方式二：本地双击

下载仓库，双击 `index.html`。完全离线可用（除了 API 调用需要联网）。

### 方式三：Python 服务器

```bash
pip install -r requirements.txt  # 实际上零依赖，标准库即可
python story_server.py
# 浏览器打开 http://localhost:8888
```

默认自带 **TD SAT 黄金单词 5.0（1765 词）**。

## 🔑 隐私

- API Key 存在浏览器 localStorage，**绝不离开你的设备**
- 词库上传后存在浏览器本地，不清缓存就不会丢
- 没有后端服务器，没有数据库，没有追踪
